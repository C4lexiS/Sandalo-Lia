# Sandalo-Lia
Sistema de Automatización de Reservas con IA (Sándalo SPA).

Descripción: Se desarrolló un ecosistema de automatizaciones en la nube para realizar tareas de envio de confirmaciones de agendamiento de citas
Stack Tecnológico: Lista n8n, Gemini 2.5 Flash, WhatsApp API, Contabo, etc.

Arquitectura: Explica brevemente cómo fluye la información (ej. Webhook WhatsApp -> n8n -> Gemini -> Google Calendar).

Cómo ver el flujo: Indica que has adjuntado el archivo .json de n8n para que puedan importar el flujo y ver la lógica de los nodos.
